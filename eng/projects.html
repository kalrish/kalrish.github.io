  • guardcheader

    #include guards, automatised

    Even in small-sized C and C++ projects, it is often necessary to protect headers with inclusion guards. This program does that in a fancy way, including adding a #line directive to make diagnostics refer to the original files.

  • json2lua

    Convert JSON to Lua

    Need to load JSON data from Lua code? No need for libraries! This program converts JSON files to Lua tables and it does that fast. Additionally, if your JSON data is static, you avoid having to ship it and parse it every time.

  • mrp

    Moral and political truth, as discovered by introspective reason

    An introspective inquiry into the roots of moral thought, the possibility of purely-rational morals and the will as well as a moral-political proposal.

    The contents, formatted in JSON, are not yet open.

    Consumer-facing versions of the book may be produced in several formats using different tools:

    • DVI (and, in turn, using dvips, PostScript) and PDF output might be produced with LuaTeX using this system.
    • An e-book in EPUB format may be generated with code which is not yet developed to a degree that it can be made public.

    Comprehensive library of font loading routines for LuaTeX

  • cxx-ucem

    C++ infrastructure and components for the Unicode Character Encoding Model

    This is a complete infrastructure designed to accommodate components abiding to the Unicode Character Encoding Model, allowing for high-performance implementations.

    The library cxx-ucem-common contains implementations of ASCII and Unicode components.

  • TeX ruled paper generator

  • TeX taster

    Convenient TeX macros to get a taste of different settings

    This code provides you with convenient macros you can use to get a taste of different typing settings (most likely, font-related settings). Embeddable and compatible with every TeX engine.
